# raucsend

```
$ raucsend install -h
runs rauc install on target, forwards embedded webserver for serving images

Usage:
  raucsend install <target> <image1> <image2> ... [flags]

Flags:
  -h, --help          help for install
  -p, --pass string   ssh password (default "root")
  -l, --user string   ssh user (default "root")

```
